\# 🏰 Belarus Journey — Великое Путешествие



\## 📖 Описание проекта



\*\*Belarus Journey\*\* — это атмосферная приключенческая игра на базе \*\*WinUI 3 (Windows App SDK)\*\*, посвященная исследованию регионов Беларуси. Игрок берет на себя роль путешественника, которому нужно изучить все области страны, управляя запасом топлива, преодолевая случайные дорожные события и побеждая в интерактивных мини-играх.



Проект сочетает в себе элементы стратегии, RPG и интерактивной карты.



\-----



\## 🔑 Основные функции



\### 🗺 Исследование и Карта:



&#x20; - \*\*Интерактивная карта:\*\* Визуализация прогресса через встроенный WebView2. Исследованные области меняют свой цвет и статус в реальном времени.

&#x20; - \*\*Система регионов:\*\* Детальный список областей с отслеживанием прогресса (в процентах) и уникальными описаниями локаций.

&#x20; - \*\*Статусная система:\*\* Продвижение от «Новичка» до «Мастера дорог» по мере накопления опыта.



\### 🎮 Игровые механики:



&#x20; - \*\*Мини-игры:\*\* Испытания на скорость реакции («Бездорожье») и внимательность («Туман»), влияющие на прогресс и ресурсы.

&#x20; - \*\*Случайные события:\*\* Система диалогов с выбором (фестивали, дорожные работы, встречи с местными жителями), которые могут как помочь, так и замедлить путь.

&#x20; - \*\*Управление ресурсами:\*\* Контроль уровня «топлива» (энергии). Механика «Привала» для восстановления сил с риском частичной потери маршрута.



\### 🎨 Технологичный интерфейс:



&#x20; - \*\*Живые анимации:\*\* Плавное заполнение полосок прогресса и всплывающие карточки игровых событий (Storyboards \& Animations).

&#x20; - \*\*Современный стек WinUI:\*\* Градиентные фоны, акриловые элементы и иконки Segoe Fluent Icons.

&#x20; - \*\*Эффекты:\*\* Визуальное празднование (конфетти) при полном исследовании региона.



\-----



\## 🛠 Технологический стек



&#x20; - \*\*Язык программирования:\*\* C++ / WinRT

&#x20; - \*\*Интерфейс:\*\* WinUI 3 (Windows App SDK 1.5+)

&#x20; - \*\*Веб-технологии:\*\* HTML5/JavaScript (для отображения и управления картой через WebView2)

&#x20; - \*\*Хранение данных:\*\* JSON (Windows.Data.Json) для сохранений

&#x20; - \*\*Архитектура:\*\* MVVM (Model-View-ViewModel)

\## 🚀 Быстрый старт



\### Требования



&#x20; - ОС Windows 10 (версия 1809 и выше) или Windows 11.

&#x20; - Microsoft Edge WebView2 Runtime.



\### Запуск игры



Приложение является портативным (\*\*Portable\*\*).



1\.  Перейдите в папку `x64/Release/BelarusJourneyGame`.

2\.  Запустите файл `BelarusJourneyGame.exe`.

3\.  Файл сохранения `savegame.json` будет создан в папке с игрой автоматически при нажатии кнопки «Сохранить».



\### Структура файлов (Release):



```bash

/BelarusJourneyGame

  ├── BelarusJourneyGame.exe   # Запуск игры

  ├── map.html                 # Файл карты (обязательно рядом с EXE)

  ├── savegame.json            # Прогресс вашего путешествия

  ├── resources.pri            # Ресурсы и локализация

  └── Microsoft.ui.xaml.dll    # Библиотеки WinUI 3

```

\## 🛠 Разработка и сборка



1\.  Откройте `BelarusJourneyGame.sln` в \*\*Visual Studio 2022\*\*.

2\.  Установите рабочую нагрузку \*\*"Разработка приложений под Windows (C++)"\*\*.

3\.  Убедитесь, что установлен компонент \*\*C++ (v143) Universal Windows Platform tools\*\*.

4\.  Соберите проект в конфигурации \*\*Release | x64\*\*.



\-----



\## ⚠️ Особенности реализации



&#x20; - \*\*DispatcherQueue:\*\* Все игровые события и таймеры синхронизированы с основным UI-потоком для стабильной работы анимаций.

&#x20; - \*\*WebView2 Bridge:\*\* Реализован мост между C++ и JavaScript для передачи данных о прогрессе на карту без перезагрузки страницы.

&#x20; - \*\*Unpackaged Mode:\*\* Игра запускается как обычный EXE-файл без необходимости установки через магазин приложений.
